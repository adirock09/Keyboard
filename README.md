# Keyboard
I wanted to make a mecanical keyboard ,So i Choose a 60% layout Keyboard as it is compact and funtional enough. We will be using switch matrix wo we need less gpio pins
## Features

- Base on Raspberry pico
- Has 61 keys ( 60% Keyboard Matrix)
